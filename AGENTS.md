# AGENTS

## Scope

This repo is the product repo for LocalLens. Keep user-facing extension assets and Chrome Web Store collateral here.

## Deployment Rules

1. Set GitHub description, homepage, and topics explicitly.
2. Keep support, privacy-policy, and reviewer-instructions links pointed at this repo.
3. Do not mix LocalLens product assets with OpenClaw operator skills.
4. Keep generated `dist/` outputs out of version control unless the user explicitly asks for checked-in release artifacts.
5. Regenerate store assets and release notes from this repo before every tagged release.

## Required Metadata

- GitHub description
- GitHub homepage
- GitHub topics
- Support URL
- Privacy policy URL
- Reviewer instructions URL
